# Changelog

## [1.0.0] - 2026-06-20

### Initial Release

- Theming for maskable graphics, text elements and selectables via scriptable objects
- ThemePalette singleton for centralized runtime preset management
- ColorReceiver, FontReceiver, SelectableReceiver components for binding scene elements
- PrefabGroupReceiver for multi-instance prefab support
- Preset switch settings: snap, lerp, sequence, cascade, random
- Theme Editor window for visual drag-and-drop binding workflow
- Color Preset Generator for extracting palettes from images
- Preset Collection editors for colors, fonts, and selectables
- Runtime API for programmatic preset switching
- Demo scenes for Colors, Fonts, Selectables, Prefabs, StressTest
